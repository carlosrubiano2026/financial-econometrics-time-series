# Financial Econometrics and Time Series Modeling

This repository contains a collection of applied projects in financial econometrics, focusing on time series modeling, volatility dynamics, and empirical analysis of financial data.

## Objective

To implement and analyze econometric models commonly used in financial applications, with an emphasis on volatility modeling, time series dynamics, and empirical validation.

## Projects

### 1. Volatility Modeling with GARCH
- Estimation of GARCH-type models
- Analysis of volatility clustering
- Model comparison and diagnostics

### 2. Time Series Modeling for Financial Data
- ARIMA-type models
- Stationarity testing and transformations
- Model selection using information criteria

### 3. Financial Metrics and Empirical Analysis
- Return calculations
- Risk metrics
- Exploratory analysis of financial time series

## Methods

- ARIMA models
- GARCH models
- Time series transformations
- Statistical inference
- Model diagnostics

## Tools

- Python
- pandas
- NumPy
- statsmodels
- matplotlib
